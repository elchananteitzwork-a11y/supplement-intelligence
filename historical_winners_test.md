# HISTORICAL WINNERS TEST
## Framework Accuracy Validation
**Date:** June 2026  
**Method:** Reconstruct pre-launch market conditions. Apply current Investment Memo framework. Output BUILD NOW / VALIDATE FURTHER / SKIP. Compare against real-world outcome.  
**Rule:** Score as if the brand does not exist. Use only market conditions known at launch date.

---

## ⚠️ IMPORTANT CAVEAT BEFORE READING

This test applies a framework built in 2026 to brands that launched between 2012 and 2021.

The framework's Virality dimension is TikTok-centric. TikTok did not exist as a commerce platform until ~2019. Brands that launched before 2020 will therefore receive systematically lower virality scores — not because they had bad viral potential, but because the **scoring dimension itself didn't exist yet at the time of their launch.**

This is not a trivial flaw. It is a structural one. Results below expose this explicitly.

## Interpretation

The framework has a **7.7% true positive rate** for BUILD NOW — meaning only 1 in 13 successful brands would have been correctly flagged to build immediately.

However, the framework has an **84.6% true-positive rate for "don't skip"** — only 2 brands received SKIP recommendations that became significant brands. Both (Liquid IV, Cymbiotika) are category-creation businesses that are structurally undetectable by a search-volume-based scoring model.

**The average score of successful brands is 64.8.** The current BUILD NOW threshold is 75. This 10-point miscalibration is the primary systemic failure. If the threshold were 65, the BUILD NOW rate would be ~54% (7/13 brands), which is much more accurate.

**The framework's core failure mode is over-conservatism, not over-optimism.** It generates few false positives but many false negatives.

## Why the Framework Systematically Misses Winners

### Failure Type 1: Era Bias (3–4 brands affected)
Nutrafol, Ritual, Hims, Hers all launched before TikTok existed as a commerce platform. Their virality scores (3–4/10) are technically accurate for 2016–2018 but create a systematic penalty for any pre-2020 brand.

### Failure Type 2: Category Creation (2 brands: Liquid IV, LMNT)
Demand scores require existing search volume. Category creators invent demand that doesn't yet exist. The framework is structurally blind to this.

### Failure Type 3: Disruption Within Crowded Market (3 brands: Seed, Ritual, Bloom)
The competition score penalizes crowded markets uniformly. Seed entered the crowded probiotic market and won by creating the premium tier. Ritual entered the crowded multivitamin market and won by creating ingredient transparency. The framework doesn't distinguish between "crowded with no gap" and "crowded with a premium gap."

### Failure Type 4: Founder Distribution (1–2 brands: Bloom, LMNT)
Both succeeded significantly because of founders with existing audiences (Mari Llewellyn's fitness following, Robb Wolf's podcast audience). Zero CAC for early customers = dramatically different unit economics than the framework assumes. No scoring dimension captures this.

### Failure Type 5: Business Model Mismatch (2 brands: Hims, Hers)
Hims and Hers are not supplement brands — they are healthcare delivery platforms. Applying a supplement scoring framework to a telehealth business produces structurally incorrect results.
